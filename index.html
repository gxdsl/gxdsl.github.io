发表于2022-11-07
Keil出现警告：last line of file ends without a newline出现warning: #1-D: last line of file ends without a newline的警告可能是Keil5的一个bug。解决方法为：在程序的最后新换一行即可。如果换行之后，依然出现此种警告，就要看你换行之后是否是全新的一行 （不能是空格！！！） Text Completion无法设置自动补全在keil编程时有些变量名会很长，如果手打的话很浪费时间而Keil5具有自动补全功能，也就是输入变量前几个字母，自动联想整个变量。但是我们进入Edit-Configuration-Text Completion时，发现本来可以设置的自动补全功能是空白一片。 不需要重新安装keil，只需要替换一些就可以了。将MDK.EXE运行并安装后，我们进入UV4文件夹,把UV4文件夹里面的UV4.exe文件进行复制即可。重新打开软件，我们会发现本来空白的地方已经有了，输入3个字母后keil就会自动产生联想，上下方向键选择，Enter/Tab键进行确认，这样就可以快速进行编程了。
